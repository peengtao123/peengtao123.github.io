Docker 入门指南
Date: 1/1/2024Category: Docker, 容器化Tag: Docker, 容器, 虚拟化, DevOps

什么是 Docker？

Docker 是一个开源的容器化平台，可以将应用程序及其依赖打包到一个可移植的容器中。容器是轻量级的虚拟化解决方案，与传统虚拟机相比，具有更快的启动速度和更低的资源消耗。

Docker vs 传统虚拟机

  • 传统虚拟机: 需要完整的操作系统，资源消耗大，启动慢
  • Docker 容器: 共享宿主操作系统内核，轻量级，启动快速
